# Imagens personalizadas i9TMG

Esta pasta recebe as imagens reais usadas pelo site. O Vite publica tudo que fica dentro de `public`, entao estes arquivos ficam acessiveis pela URL `/i9-user-images/nome-do-arquivo`.

O sistema usa estas imagens locais primeiro. Se voce trocar qualquer arquivo mantendo o mesmo nome, o site atualiza no proximo build/deploy.

## Como usar

1. Troque a imagem nesta pasta mantendo exatamente o mesmo nome.
2. Rode `npm run dev` ou faca o deploy.
3. Abra a tela correspondente e recarregue o navegador.

Para trocar o logo principal, continue usando `public/logo-i9tmg.png`.
